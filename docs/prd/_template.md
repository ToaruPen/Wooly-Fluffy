## 禁止語辞書（参照用）

以下の表現は使用禁止。具体的な表現に置き換えること。

- 適切に: 「〜の条件を満たす場合」
- なるべく: 「〜以上/以下」
- できるだけ: 「最大〜まで」「最小〜」
- 高速: 「〜秒以内」「〜ms以内」
- 簡単に: 「〜ステップで」「〜クリックで」
- ユーザーフレンドリー: 具体的なUI要件を列挙
- 柔軟に: 「〜の場合は〜、〜の場合は〜」
