# /codex-pr-review

Legacy compatibility command.

This repository migrated to `/pr-bots-review` as the canonical review-bot workflow.

## Usage

```text
/codex-pr-review <PR_NUMBER_OR_URL>
```

## Behavior

- Treat this command as an alias to `/pr-bots-review`.
- Set `AGENTIC_SDD_PR_REVIEW_MENTION='@codex review'` when targeting Codex.
- Follow `.agent/commands/pr-bots-review.md` for the full and up-to-date procedure.
